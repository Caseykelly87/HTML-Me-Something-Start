Welcome to your first Pro Tools® session

Today we are going to record your first ever song on Pro Tools. This turorial does make some assumtions, for instance, You have already installed the software have the input output settings working with your microphone and chosen "monitors", whether they are headphones or speakers. Keep in mind that you cannot have your microphone and speakers active at the same time if they are in the same space. We will also assume that you are recording vocals for an instrumental file that's already accessible on your computer, preferably a .WAV for best audio quality.

Now that your studio is ready let's look at how to create a new project and start recording! You can click on the numbered the tasks to jump to that point, most tasks have multiple steps.

There is also a two minute video tutorial at the bottom of this page that shows the process in action!

How to record your first song

  1. Create new session
    • Name your session
    • Confirm session parameters
  2. Import Audio
    • Locate audio file
    • Convert audio
  3. Create new Tracks
    • Select new track settings
  4. Enable recording
    • Enable track recording
    • Enable session recording
  5. Record!

Create your new session

Enter the name of your project into the Name: field. Now ensure that the set parameters for: File Type, Sample Rate, Bit Depth and I/O Settings are appropriate for your project. Don't forget to verify that the Location where the file will be saved is correct before clicking Create.


This is your empty session. Next let's add the instumental, unless your signing acapella today?


Import Audio

Navigate to File in the top left corner of the screen, go down the menu list and hover on Import a sub-menu will open, click Audio in that window.

Using the Look in: bar at the top of the Import Audio window locate the desired audio file.

Before you select the file that you wish to import. Notice the two large empty fields on this window. Both empty fields need to contain the file intended for import. If you double click on your audio file it will automatically convert and occupy both fields, otherwise use next step to convert.

Convert the file by selecting it in middle window and clicking Convert Clip.

Now that the file is converted you should see the file name three times total, one in each window, it's time to import. Click Done.

Set the destination for the converted audio to be stored. It should default to the Audio Files folder of the session. Be sure to click the Use Current Folder button NOT the Open button

Choose New Track option and click OK.

Here is your session with the instrumental track! Next let's add some empty tracks for your voice.

Create New Tracks

To create more tracks for you to record on navigate to the Track menu at the top of the page. Then click New.

Decide on the number of tracks to create and set appropriate parameters. Click Create.

Here is your session ready for vocals! Now tell pro tools where you want to record by enabling record.

Enable Recording

Press the red button with the white circle on the Audio 1 Track to set it as the destination for your new audio. Once you record somthing that you wish to keep, you would disenable Audio 1 recording and enable Audio 2. Enabling record on the track sets the destination but we have to tell the session that it's time to record.

Press the Record button with red circle on top of the screen to enable session recording. Finally your qued up and it's time.

Record!

Press the play button located next to session record button or simply press the Space Bar to start recording!
Note: Space Bar will also stop recording, as well as start and stop regular playback.

We hope this quick ProTools® tutorial was helpful, Good Luck and we"ll be sure to keep an ear out for you!
